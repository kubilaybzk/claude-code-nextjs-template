# Accessibility (a11y) — Minimum Rules

## Icon-Only Buttons

Yalnızca ikon içeren butonlarda `aria-label` zorunludur.

```tsx
// ✗ Forbidden
<Button variant="ghost" size="icon">
  <TrashIcon />
</Button>

// ✓ Correct
<Button variant="ghost" size="icon" aria-label="Şirketi sil">
  <TrashIcon />
</Button>
```

## Görsel Olmayan Yardımcı Metin

Ekran okuyucuya bilgi vermek için `sr-only` class kullanılır.

```tsx
<Button>
  <PlusIcon />
  <span className="sr-only">Yeni şirket ekle</span>
</Button>
```

## Form Alanları

Her input, `label` ile ilişkilendirilmelidir. shadcn `<FormLabel>` bunu otomatik sağlar.

```tsx
// ✓ FormLabel htmlFor otomatik bağlar
<FormItem>
  <FormLabel>Şirket Adı</FormLabel>
  <FormControl>
    <Input placeholder="Örn: Acme Ltd." {...field} />
  </FormControl>
  <FormMessage />
</FormItem>
```

## Renk Tek Başına Bilgi Vermemeli

Hata, uyarı, başarı durumları yalnızca renkle ifade edilmez — ikon veya metin de eklenir.

```tsx
// ✗ Sadece renk
<Badge className="bg-destructive">Hata</Badge>

// ✓ İkon + renk
<Badge variant="destructive">
  <AlertCircleIcon className="size-3" />
  Hata
</Badge>
```
